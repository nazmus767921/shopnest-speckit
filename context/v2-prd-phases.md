# ShopNest V2 — Phased Implementation Plan

## Phase 1: Checkout & Sales Optimization

This phase focuses on features that directly impact customer conversion and sales psychology, prioritizing checkout completion and perceived value.

### 1. Cash on Delivery (COD) Checkout Integration

* **Description:** Complete the integration of the Cash on Delivery (COD) payment method so that customers can select it during the checkout process.


* **Current State:** The feature is currently only partially implemented; a feature flag exists within the plan editor, but the actual checkout flow only supports bKash and Nagad.


* **Requirements:** COD must be explicitly surfaced and fully functional within the customer payment flow.



### 2. Compare-at-Price

* **Description:** Implement functionality that allows merchants to display an original "crossed-out" price next to the current selling price to highlight discounts.


* **Current State:** Currently not implemented.



---

## Phase 2: Discovery & Brand Customization

This phase focuses on merchant empowerment, improving product discoverability through search engines, and allowing better brand identity through visual customization.

### 1. Product SEO

* **Description:** Introduce Search Engine Optimization (SEO) capabilities at the individual product level.


* **Requirements:** The system must allow merchants to define and save a `meta_title` and a `meta_description` for each product.


* **Current State:** Currently not implemented.
