# Elegance Template Design System

## Overview
Elegance is a refined, minimalist template designed for fashion and lifestyle brands. It emphasizes negative space, large imagery, and crisp typography to create a premium feel.

## Color Rhythm
Elegance uses a strict alternating color rhythm to ensure sections flow naturally:
- **Light**: `var(--color-surface)`
- **Dark**: `var(--color-shade-20)`
- **Accent**: Configurable via theme variables

## Typography
- **Display**: Serif fonts (e.g., Playfair Display) for headlines and large text.
- **Sans**: Clean sans-serif fonts (e.g., Inter, Helvetica) for body text and UI elements.

## Component Style
- **Buttons**: Square or slightly rounded (`radius: none` or `radius: sm`), uppercase tracking, bold text.
- **Cards**: Minimal borders, full-bleed images, subtle hover effects.

## Layout Constraints
- Max width: `wide` (typically 1440px or full-bleed)
- No sidebar.
